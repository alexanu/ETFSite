This site will convert a portfolio containing ETFs and break it down into exposure to their individual holdings.

Instructions:
  • Go to Yahoo Finance in another tab and sign in.
  • Select a portfolio which contains ETFs and click on download as CSV.
  • Once the CSV file has been downloaded, upload it here using the File Uploader.
  • Wait a moment, and the page will be redirected to your updated CSV with a download link.
Notes:
***This script will only work with US or Canadian equities, and will return holdings in $CAD
***It can only return the top 25 holdings, any remaning value will be added back to the ETF ticker with a 'Misc' notation {{ dropzone.create('/') }} {{ dropzone.load_js() }} {{ dropzone.config() }}
